2.1.2 - Headline Klassen

Die Überschriften klassen .h1 bis .h6 erlauben es Elemente optisch wie eine Überschrift anzupassen. Überschriften-Elemente werden durch die Klassen überschrieben, so dass eine <h1 class="h3"> wie eine Überschrift der dritten Ebene aussieht.

  • (default)

    Sieht nur aus wie eine Überschrift

  • .h1

    Sieht aus wie eine H1

    Sieht nur aus wie eine Überschrift

  • .h2

    Sieht aus wie eine H2

    Sieht nur aus wie eine Überschrift

  • .h3

    Sieht aus wie eine H3

    Sieht nur aus wie eine Überschrift

  • .h4

    Sieht aus wie eine H4

    Sieht nur aus wie eine Überschrift

  • .h5

    Sieht aus wie eine H5

    Sieht nur aus wie eine Überschrift

  • .h6

    Sieht aus wie eine H6

    Sieht nur aus wie eine Überschrift
