1.0 - Buttons

Die Standardklasse für buttons ist .button und wird durch modifier erweitert.

  • (default)
    Link
  • :hover

    Zustand: hover

    Link
  • :disabled

    Zustant: disabled

    Link
  • .button--primary

    Hauptbutton (primary)

    Link
  • .button--secondary

    Sekundärbutton (secondary)

    Link
  • .button--small

    Kleiner Button (einzeilig)

    Link 
  • <button class="button {$modifiers}">Button</button>
<a href="#" class="button {$modifiers}">Link</a>
<input type="button" value="input[type='button']" class="button {$modifiers}">
